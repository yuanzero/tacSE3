# TacSE3 — Project Page

Project page for *TacSE3: Equivariant SE(3) Motion Estimation from Low-Texture Visuotactile Images for In-Gripper Tracking and Compensation* (Under Review, 2026).

## Run locally

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy

Push to `main`, then enable GitHub Pages (Settings → Pages → Branch `main` / root).

## Layout

```text
index.html          # single-page site
style.css
script.js           # BibTeX copy + nav scroll-spy
assets/
  images/           # overview.png, diffobj.png, setup.png
  videos/           # demo.mp4, rotation_demo/, generalization_demo/, *_demo_*.mp4
```
